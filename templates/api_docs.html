API Documentation

Authentication

All API requests require authentication using your API key:

Authorization: Bearer {{ api_key }}

Or as a query parameter:

?api_key={{ api_key }}
Consumer Behavior Dataset

Detailed consumer purchase patterns across regions

Endpoint:
GET /api/v1/consumer-behavior
Example Request:
curl -H "Authorization: Bearer {{ api_key }}" https://yourdomain.com/api/v1/consumer-behavior
Example Response:
{
    "product": "Consumer Behavior Dataset",
    "endpoint": "consumer-behavior",
    "data": [
        {
            "customer_id": "a1b2c3d4",
            "age": 35,
            "income_bracket": "medium",
            "purchase_category": "electronics",
            "purchase_amount": 299.99,
            "region": "California",
            "date": "2023-05-15"
        },
        ...
    ]
}
Financial Market Trends

Daily stock market indicators and trends

Endpoint:
GET /api/v1/financial-trends
Parameters:
  • symbol - Stock symbol (optional)
  • start_date, end_date - Date range (optional)
Example Response:
{
    "product": "Financial Market Trends",
    "endpoint": "financial-trends",
    "data": [
        {
            "date": "2023-05-15",
            "symbol": "AAPL",
            "open_price": 145.67,
            "close_price": 147.89,
            "volume": 4567890,
            "market_cap": 2500000000
        },
        ...
    ]
}
Weather History API

Historical weather data for global locations

Endpoint:
GET /api/v1/weather
Parameters:
  • location - City name (optional)
  • start_date, end_date - Date range (optional)
